# Follow-ups — готовые шаблоны для агентов

Короткий файл с копируемыми скриптами Day1 / Day3 / Day7, ветвлениями и операционными правилами.

---

## Общая инструкция
- Всегда внеси кандидата в CRM ПЕРЕД отправкой реф‑ссылки.
- Отвечай голосом на вопросы (15–40 с), когда это возможно — это повышает доверие.
- Два аккуратных пинга: Day3 и Day7; затем пометка для перезапуска через 2–3 недели.

--

**Вопросы / поддержка:** если нужна помощь от команды — пиши в Telegram `@jobwith_huntme` одним сообщением по форме:
```
Я агент: [СОЛО|КОМАНДА]
Имя: [Ваше имя]
Telegram: [@username]
Коротко: [вопрос/проблема в 1-2 строки]
Активен: [да/нет]
```
Не отправляй дубли — после проверки тебя добавят в папку и ответят.

## День 1 — Основное сообщение (скрипт для копирования)

```
Привет, [имя]! Меня зовут [имя агента], я помогаю людям находить удалённую подработку — подбор кандидатов для партнёров HuntMe. Это гибкая подработка без вложений: короткое обучение, поддержка и возможность получить первые выплаты уже после первой модели. Подходит для совмещения с основной работой или для тех, кто хочет дополнительный доход. Многим хватило 30–60 минут в день, чтобы начать. Посмотри короткий ролик (3 мин) в боте (он доступен в меню). Актуален ли для тебя поиск работы сейчас и на что ты обычно обращаешь внимание?
```

- Цель: вывести кандидата на ответ (диалог), узнать его потребности.

---

## Ветвления — ответы кандидата

### 1) "Интересует / Хочу"

```
Отлично — пришли, пожалуйста, ФИО, Telegram и телефон. Я внесу в CRM и пришлю ссылку на регистрацию: [вставьте свою персональную реф‑ссылку из бота].
```

- Действия: внести в CRM → отправить свою персональную реф‑ссылку из бота.

### 2) "Вопросы" (кандидат задаёт уточнения)
- Рекомендация: ответить голосом (15–40 с). Если голосом невозможно — коротким текстом.
- Заканчивай ответ вопросом: "Хочешь, я прямо сейчас пришлю регистрацию?"

Пример короткого текста:
```
Понимаю, вопрос по графику — работа гибкая, можно по вечерам. Хочешь, дам короткий чек‑лист, что делать в первые 2 дня?
```

### 3) "Неинтересно" / отказ

```
Понял(а). Можно уточнить — что именно не подошло (время / формат / сомнения в легальности / другое)?
```

- После ответа: попробуй обработать причину; если кандидат остаётся неинтересен — пометить "неинтересно" и поставить напоминание на повторный контакт через **2–3 недели**; если и после этого не будет отклика — планируем мягкий пинг через 1–2 месяца.

## Если нет ответа (игнор)

- День 3 (альтернативная подача — другая подача УТП):
```
Привет, [имя]. Возможно, занят(а) — коротко: у нас можно начать за 1–2 дня, задачи простые. Сколько у тебя свободного времени в неделе? Могу предложить вариант под твой график.
```

- День 7 (финальный пинг):
```
Пишу финально — если позже будет удобно, дай знать, напомню. Могу прислать 1‑минутный ролик, чтобы не тратить много времени.
```

- Действие: после двух пингов пометить «перезапуск через 2–3 недели» в CRM.

## Скрипты возражений (копировать)

- "Нет времени" →
```
Понимаю, можно начать с 30 мин в день; Обучение бесплатное-график свободный хочешь вариант под вечер?
```

- "Сомнения в легальности" →
```
Это стандартный HR/подбор — без вложений. В дальнейшем в партнерке ты увидишь-пообщаешься с активными агентами и убедишься во всем сам/ запросишь пример выплат.
```

- "Хочу подробности о деньгах" →
```
Средний результат новичка — от $500/месяц, но точнее зависит от объёма; в большинстве случаев новички работают по базовым офферам-выходят на результат за 1-2 неделю
```

## Быстрый cheat‑sheet (2 строки для карточки)
- День1: коротко > ролик 3м > вопрос в конце (актуально ли). 
- Day3: альтернативная подача + вопрос про свободное время. Day7: финальный пинг. Всегда CRM → ссылка.
